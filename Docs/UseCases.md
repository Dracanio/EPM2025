# Use Cases

1. Poster erstellen
2. Poster bearbeiten
3. Poster speichern
4. Poster exportieren
5. Assets hochladen
6. Vorlage auswählen
7. Poster teilen
8. Projekt verwalten
   8.1 Projekt erstellen 
   8.2 Collaboratoren einladen
   8.3 Projekt Fertigstellen / Archevieren
9. Vorlage erstellen
10. Vorlage veröffentlichen 


# Workflow

1. User -> Erstellt ein Poster -> wählt eine Vorlage aus -> bearbeitet das Poster -> teilt das Poster -> exportiert das Poster  
2. User -> wird zu Projekt als collaborator eingeladen -> Bearbeitet das Poster
3. User -> erstellt eine Vorlage -> veröffentlicht diese
4. User  -> lädt Asset hoch -> fügt dieses in sein Poster ein
5. User  -> Erstellt ein Poster -> wählt eine Vorlage aus -> bearbeitet das Poster -> Speichert das Poster -> Pause -> öffnet erneut und bearbeitet weiter
